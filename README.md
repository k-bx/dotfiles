Dotfiles
========

My emacs configuration. Maybe something else in future.

- Bitbucket: https://bitbucket.org/k_bx/dotfiles
- Github: https://github.com/k-bx/dotfiles

Usage
-----

After cloning, do:

```
cd dotfiles/emacs.d
cask install
cask update
cd ..
ln -s dotemacs.el ~/.emacs
ln -s emacs.d ~/.emacs.d
```
